## Introduce
해당 프로젝트는 React-Native_Expo 프레임워크를 사용해 개발한 NativeApp입니다.
서버는 node.js의 Express 프레임워크를 사용했습니다.

## Features
- 메인 상품페이지
- 메인 Carousel 구현
- 상품 상세페이지
- Post method를 활용한 상품 구매
- Update를 활용한 품절 기능
- 리뷰 페이지
